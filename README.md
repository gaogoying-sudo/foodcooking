# 菜谱可视化系统 Recipe Visualizer

## 📊 版本说明

| 版本 | 功能 | 文件 |
|------|------|------|
| V1 | 极速版 | index.html |
| V2 | 维度分析 | index-dimensions.html |
| V3 | 功能完整 | index-v3.html |
| V4 | 集成版 | index-v4.html |
| V5 | 虚拟滚动 | index-v5.html |
| V6 | SOP 时间轴 | index-v6.html |

## 🚀 部署到 Vercel

```bash
# 1. 安装 Vercel CLI
npm install -g vercel

# 2. 登录 Vercel
vercel login

# 3. 部署
vercel --prod
```

## 📁 数据结构

- `data/recipes.json` - 示例数据（5 条）
- `data/recipes-full.json` - 完整数据（36 条）
- `data/processed/recipe-profiles.json` - V6 标准数据（13,510 条）
