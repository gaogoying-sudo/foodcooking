## 📊 界面功能

### 左侧：菜谱列表
- 显示所有导入的菜谱卡片
- 显示菜谱名称、ID、时长、功率
- 数据质量状态（正常/警告/异常）
- 支持搜索筛选

### 右侧：时间轴展示
- **功率曲线图** - SVG 绘制功率变化
- **步骤时间轴** - 按时间顺序展示所有步骤
  - 🔵 蓝色：设备控制（功率/搅拌）
  - 🟢 绿色：手动投料
  - 🟡 黄色：自动投料
- **步骤详情** - 点击步骤查看完整信息

### 顶部操作栏
- 📁 导入菜谱数据 - 上传 JSON 文件
- 📊 加载示例数据 - 加载内置示例
- 🔗 飞书表格同步 - 配置 API 直连
- 💾 导出数据 - 导出当前数据
