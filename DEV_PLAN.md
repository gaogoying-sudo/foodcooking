## 🔧 技术架构

### 前端技术栈
- **核心:** HTML5 + CSS3 + Vanilla JavaScript
- **图表:** SVG 手绘（无依赖）
- **存储:** localStorage + IndexedDB
- **构建:** 无需构建工具（纯静态）

### 后端接口（可选）
- **飞书 API 代理:** Node.js + Express
- **数据缓存:** Redis（可选）
- **文件服务:** 本地/云存储
